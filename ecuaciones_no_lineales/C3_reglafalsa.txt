Pseudocódigo para el algoritmo correspondiente al método de la regla falsa en el cual se buscan raíces de una ecuación 
a partir de un intervalo dado.

Data: f (función), a (valor izquierdo), b (valor derecho), tol (tolerancia), i (iteraciones)
Result: raíz sin error r, raíz con error(aproximado) ri  o mensaje de fallo

Leer a, b, tol, i 
Ya <- f(a)
Yb <- f(b)
if Ya = 0 then
    Mostrar a es raíz
else 
    if Yb = 0 then
        Mostrar b es raíz
    else
        if Ya * Yb > 0 then
            Mostrar el intervalo no es valido 
        else 
            p <- a - ((f(a)*(b-a))/(f(b)-f(a)))
            Yp <- f(p)
            error <- tol + 1
            contador <- 1
            while Yp != 0 ∧ error > tol ∧ contador < i do 
                if Ya * Yp > 0 then
                    a <- p
                    Ya <- Yp
                else
                    b <- p
                    Yb <- Yp
                paux <- p 
                p <- a - ((f(a)*(b-a))/(f(b)-f(a)))
                Yp <- f(p)
                error <- |p - paux| v |(p - paux)/p|
                contador <- contador + 1
            if Yp = 0 then  
                Mostrar p es raíz
            else
                if error <= tol then
                    Mostrar p es una raíz con un error igual a error
                else
                    Mostrar El metodo falló
